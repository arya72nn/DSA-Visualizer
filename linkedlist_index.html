Linked List

Dynamic Data Structure for Efficient Memory Utilization

What is a Linked List?

A Linked List is a linear data structure where elements are stored in nodes. Each node contains two parts: data and a reference (link) to the next node in the sequence. Unlike arrays, linked lists do not store elements in contiguous memory locations.

Types of Linked List

  • Singly Linked List – Each node points to the next node
  • Doubly Linked List – Nodes have both next and previous pointers
  • Circular Linked List – Last node points back to the first node

Basic Operations

  • Insertion (Beginning, End, Specific Position)
  • Deletion (Beginning, End, Specific Node)
  • Traversal
  • Searching
  • Reversing the Linked List

Advantages

  • Dynamic size (no fixed length)
  • Efficient insertion and deletion
  • No memory wastage

Disadvantages

  • Extra memory for pointers
  • No direct access like arrays
  • Traversal is slower than arrays

Time Complexity

  • Access: O(n)
  • Search: O(n)
  • Insertion: O(1)
  • Deletion: O(1)
