Graph

Non-linear data structure for connected data

Introduction

A Graph is a non-linear data structure consisting of vertices (nodes) and edges that connect pairs of nodes. Graphs are used to represent networks and relationships.

Graph Terminology

  • Vertex (Node)
  • Edge
  • Directed & Undirected Graph
  • Weighted & Unweighted Graph
  • Degree of Vertex

Types of Graphs

  • Undirected Graph
  • Directed Graph (Digraph)
  • Weighted Graph
  • Cyclic & Acyclic Graph
  • Connected & Disconnected Graph

Graph Representation

  • Adjacency Matrix
  • Adjacency List

Graph Traversals

  • Breadth First Search (BFS)
  • Depth First Search (DFS)

Applications of Graph

  • Social Networks
  • Google Maps & GPS
  • Computer Networks
  • Web Page Ranking
  • Routing Algorithms
