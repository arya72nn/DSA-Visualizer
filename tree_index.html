Tree

Hierarchical non-linear data structure

Introduction

A Tree is a non-linear data structure used to represent hierarchical relationships between elements. It consists of nodes connected by edges and has a root node.

Tree Terminology

  • Root
  • Parent & Child
  • Leaf Node
  • Subtree
  • Depth & Height

Types of Trees

  • Binary Tree
  • Binary Search Tree (BST)
  • AVL Tree
  • Heap
  • N-ary Tree

Tree Traversals

  • Inorder Traversal
  • Preorder Traversal
  • Postorder Traversal
  • Level Order Traversal

Time Complexity (BST)

  • Insertion: O(log n)
  • Deletion: O(log n)
  • Searching: O(log n)

Applications of Tree

  • File System
  • Database Indexing
  • Decision Trees
  • Compiler Design
  • Routing Algorithms
