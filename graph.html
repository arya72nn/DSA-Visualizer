Graph Visualization

Add nodes, connect edges, and visualize traversals

Instructions

  • Add nodes with unique names or numbers.
  • Drag & fix node positions before creating edges.
  • Create edges: specify From Node, To Node, and Weight if weighted.
  • Select Directed/Undirected and Weighted/Unweighted types.
  • Use DFS or BFS to visualize traversal.
  • Green node → Visited, Yellow → Currently processing.
  • Click Clear Graph to reset.
