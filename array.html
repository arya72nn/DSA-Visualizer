Array Traversal

Access all array operations in one screen

Instructions

  • Enter array elements separated by commas (e.g., 5,10,15,20).
  • Click Traverse to visit elements left to right.
  • Click Reverse to visit elements right to left.
  • Click Max & Min to highlight max (red) and min (green).
  • Click Linear Search to find a value.
  • Enter value & position to Insert an element.
  • Enter position to Delete an element.
  • Click Bubble Sort for ascending sort animation.
  • All elements are visualized as colored boxes.
