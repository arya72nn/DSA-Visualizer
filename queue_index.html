Queue

Linear data structure based on FIFO principle

Introduction

A Queue is a linear data structure that follows the FIFO (First In First Out) principle. The element that is inserted first is removed first.

Characteristics of Queue

  • Follows FIFO (First In First Out)
  • Insertion happens at the rear
  • Deletion happens from the front
  • Sequential access only

Types of Queue

  • Simple Queue
  • Circular Queue
  • Priority Queue
  • Deque (Double Ended Queue)

Basic Queue Operations

  • Enqueue (Insertion)
  • Dequeue (Deletion)
  • Peek / Front
  • IsEmpty
  • IsFull

Time Complexity

  • Enqueue: O(1)
  • Dequeue: O(1)
  • Peek: O(1)

Applications of Queue

  • CPU Scheduling
  • Printer Spooling
  • BFS Traversal in Graph
  • Task Scheduling
  • Order Processing Systems
