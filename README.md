# 📊 DSA Visualizer

An interactive **Data Structures and Algorithms (DSA) Visualizer** built to help students and developers understand complex concepts through animations and step-by-step execution.

---

## 🚀 Features

* 🔍 Visualize popular algorithms in real-time
* 📚 Supports multiple data structures
* ⚡ Step-by-step execution with clear transitions
* 🎨 Clean and user-friendly UI
* 🧠 Helps in better understanding of logic and flow

---

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript

---

## 📌 Algorithms Included

* 🔄 Sorting Algorithms

  * Bubble Sort

* 🌳 Data Structures

  * Arrays
  * Linked List
  * Stack
  * Queue
  * Trees (Binary Tree, BST)

* 🔍 Searching Algorithms

  * Linear Search
